# 8086-digital-clock
8086 Digital Clock is an Assembly Language project developed using EMU8086 that displays a real-time digital clock using low-level programming concepts.
## Overview

This project is a real-time digital clock implemented using 8086 Assembly Language with GUI Turbo Assembler.

The clock continuously displays and updates the system time using low-level programming concepts.

## Features

- Real-time clock display
- Time updating mechanism
- Assembly-level implementation
- DOS-based execution
- Low-level system interaction

## Technologies Used

- 8086 Assembly Language
- GUI Turbo Assembler
- DOS Environment

## How It Works

The program retrieves system time values and continuously updates the digital clock display on the screen.

Through this project, I learned:

- Assembly language programming

- Interrupt handling basics
- Time management routines
